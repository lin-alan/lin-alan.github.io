# Fridge Tracker

A simple local fridge tracker.

## Features

- Add ingredients with expiry dates
- Highlight items that are expiring soon or already expired
- Show recipe ideas based on current fridge items
- Data saved in Supabase

## Getting started

1. Open `fridge-tracker/index.html` in your browser.
2. Add ingredients and expiry dates.
3. See what can be cooked from your current items. (No AI, just manually input for now)
