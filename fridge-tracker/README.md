# Fridge Tracker

A simple local fridge tracker app for GitHub Pages.

## Features

- Add ingredients with expiry dates
- Highlight items that are expiring soon or already expired
- Show recipe ideas based on current fridge items
- Save data locally in the browser

## Getting started

1. Open `fridge-tracker/index.html` in your browser.
2. Add ingredients and expiry dates.
3. See what can be cooked from your current items.

## Deployment

Place the `fridge-tracker` folder in your GitHub Pages repository and open `fridge-tracker/index.html`.
